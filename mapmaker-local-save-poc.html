About this demo

This is a concept demo. Draw features, give each one a name, a note, and a photo, then use Save to file to download your map as a .mmap file on this computer. Reopen it any time with Open file… — no sign-in or online account needed. Your photos are saved inside the file, so nothing is stored in an ArcGIS account.
